# OPEO - Site Officiel

Site web officiel de OPEO Consulting - Stratégie opérationnelle et transformation d'entreprise.

## 📋 À propos

OPEO est un cabinet de conseil spécialisé dans les opérations stratégiques et la transformation d'entreprise. Notre site présente nos secteurs d'expertise, notre approche et nos publications.

## 🚀 Démarrage rapide

### Prérequis
- Node.js 14+ ou Python 3.6+

### Installation et lancement

#### Option 1 : Avec Node.js
```bash
npm install
npm start
# Le site sera accessible à http://localhost:3030
```

#### Option 2 : Avec Python
```bash
python3 -m http.server 8000
# Le site sera accessible à http://localhost:8000
```

## 📁 Structure du projet

```
opeo-site/
├── index.html                    # Page d'accueil
├── a-propos.html                # Page À propos
├── contact.html                 # Page Contact
├── mentions-legales.html        # Mentions légales
├── transformation.html          # Page Transformation
├── expertise-*.html             # Pages expertise (Stratégie, Digital, R&D, Supply Chain)
├── secteur-*.html               # Pages secteurs (Aéronautique, Pharma, Luxe, Process)
├── article-example.html         # Exemple d'article
├── styles.css                   # Feuille de styles principale
├── main.js                      # Scripts JavaScript
├── server.js                    # Serveur Node.js (optionnel)
├── assets/                      # Ressources
│   └── images/                  # Images du site
└── deploy/                      # Configuration de déploiement
```

## 🎨 Pages disponibles

- **Accueil** - `index.html` - Page d'accueil avec présentation

- **Secteurs d'expertise** - Pages individuelles par secteur
- **Expertises** - Pages détaillées des domaines de compétence
- **Transformation** - `transformation.html` - Approche de transformation

## 🌐 Déploiement

### Sur GitHub Pages
1. Créer un repo `opeo-site` sur GitHub
2. Activer GitHub Pages dans les paramètres
3. Déployer la branche `main`

### Sur un serveur web
- Copier tous les fichiers sur un serveur web statique
- Utiliser un serveur Node.js ou Python pour le développement local

## 🛠️ Développement

### Modification des styles
Éditer `styles.css` pour modifier l'apparence du site.

### Modification du contenu
Les fichiers HTML peuvent être édités directement pour modifier le contenu.

### Ajouter une nouvelle page
1. Créer un nouveau fichier HTML
2. Utiliser le template des autres pages
3. Ajouter les liens de navigation si nécessaire
